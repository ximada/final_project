# Pseudocodigo "Cifrado Cesar"

1. Crear 2 funciones llamadas, cipher y decipher
2. Crea un "prompt" que pida al usuario ingresar una frase
3. Verificar que el texto sea; numero o letra.
4. Si el  usuario ingrese un campo  vacio lanzar un "alert" de error
5. Recorrer el String por medio de un "for"
6. Comparar por medio de un "if" para saber si son maysuculas o minuscula
7. Encontra el numero de la letra en ACII por medio de charCodeAt(i) y String.fromCharCode()
8. Desarrolla la formula (= -65 + 33)% 22+65
8. Regresar el mensaje cifrado
